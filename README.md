# smartstork

SmartStork is a user-friendly Progressive Web App (PWA) built with React.js and TypeScript, designed for efficient learning of foreign-language vocabulary. It helps users develop and strengthen their language skills through structured, interactive study sessions, making it ideal for self-paced learners, students, and anyone looking to expand their linguistic repertoire.

v2.0.0 The application is currently in testing and available to a closed circle of users.

--== Current Features
✅ Import word lists from simple text/CSV files
✅ Group what you want to study
✅ Interactive quiz-game mode
✅ Simple progress tracking
✅ Multiple quiz types: basic multiple choice, type-the-answer with on-screen keyboard, flip cards
✅ Word management: add, edit, delete
✅ Export/import backups
✅ Mobile-app style interface

--== Planned Features
-- Detailed study statistics (words learned)
-- Settings for quiz
-- Dark theme
-- Server synchronization between desktop and mobile versions
-- Audio playback
